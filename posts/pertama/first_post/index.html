1 Parts and components

Penggunaan Flip-Flop dan gerbang logika dalam perancangan sistem lampu lalu lintas memberikan dasar yang kuat untuk memahami konsep Finite State Machine (FSM) dan logika sekuensial. D Flip-Flop, sebagai elemen penyimpanan data 1-bit, dapat dimanfaatkan untuk mengontrol transisi state, sementara gerbang AND berfungsi sebagai pengatur kondisi.

2.1 Set the IC555 in to Astable Mode

Pertama kita set terlebih dahulu ic555 sebagai timer clock, timer clock adalah Timer clock atau pengatur waktu adalah alat yang menghitung mundur dari durasi waktu yang ditentukan, dan berhenti ketika mencapai nol, Astable mode adalah Mode astabil pada timer 555 adalah konfigurasi di mana rangkaian tidak memiliki kondisi stabil, dan terus menerus menghasilkan sinyal pulsa persegi panjang (gelombang kotak) tanpa henti. nah dalam ic555 ada fitur tersebut untuk dapat menggunakan fitur nya kita butuh

  1. kapasitor = 330uf, 1uf
  2. Resistor = 5k (2 buah)
  3. Kabel Jumper
  4. Baterai 5V

Dalam projek ini kami kami menggunakan IC 555 sebagai timer clock dengan membuat mengkonfigurasinya ke Astable

Mode yang berarti IC 555 akan terus- menerus menghasilkan gelombang persegi

(high–low) tanpa memerlukan trigger eksternal. Mode ini sering disebut sebagai free-running oscillator. Frekuensi dan duty cycle gelombang output diatur oleh dua resistor (R1 dan R2) dan satu kapasitor (C) eksternal yang akan menentukan berapa jeda waktu perpindahan lampu traffic light. 3. Conclusion

  • Berdasarkan hasil perancangan dan analisis yang telah dilakukan, dapat disimpulkan bahwa sistem lampu indikator tiga keadaan (merah, kuning, hijau) berhasil dibuat menggunakan dua buah D Flip-Flop, satu gerbang AND, dan rangkaian osilator IC 555 dalam mode astable sebagai sumber clock.
  • Semua dikontrol dari dua D Flip-Flop yang saling terhubung secara logis.
  • Siklus terbentuk dengan bantuan umpan balik dari Q dan Q̅.
  • AND gate menentukan kapan siklus “di-reset” kembali ke awal.
